# Tripleten web_project_around_es

### Descripción del Proyecto

"Around The U.S." es una página web interactiva diseñada como una galería de fotos donde los usuarios pueden compartir imágenes de lugares interesantes. El proyecto se centra en la creación de una interfaz dinámica utilizando JavaScript, permitiendo a los usuarios personalizar su perfil y gestionar una colección de tarjetas visuales.

### Funcionalidad

- **Edición de Perfil:** Los usuarios pueden actualizar su nombre y descripción mediante un formulario emergente (modal).
- **Gestión de Tarjetas:**
  - Visualización de tarjetas iniciales cargadas dinámicamente desde un array.
  - Añadir nuevas tarjetas personalizadas con nombre y enlace de imagen.
  - Eliminar tarjetas existentes.
  - Sistema de "Me gusta" (Like) interactivo en cada tarjeta.
- **Visualizador de Imágenes:** Al hacer clic en cualquier imagen de la galería, esta se abre en una ventana emergente a pantalla completa con su título correspondiente.
- **Interfaz Responsiva:** El diseño se adapta a diferentes tamaños de pantalla (móvil, tablet y escritorio) siguiendo la metodología BEM.

### Tecnologías Utilizadas

- **HTML5:** Estructura semántica y uso de etiquetas `<template>` para renderizado dinámico.
- **CSS3:** Diseño responsivo, Flexbox, Grid Layout y metodología BEM (Block Element Modifier).
- **JavaScript (Vanilla):** Manipulación del DOM, manejo de eventos, funciones de flecha y lógica de modales reutilizable.
